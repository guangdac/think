简易云文档管理系统

Document Management System

简介

简述：能实现本地文档上传到云盘存储或者从云盘下载到本地的操作，并实现对文档管理的日常操作。
集中存储: 云盘管理
目录结构: 树状层次结构
显示模式: 列表模式
全文搜索: 按文件名关键字搜索
权限管理: 提供权限控制机制(TBD)
1. 上传本地文档到云盘（暂定为七牛云的存储技术）
2. 从云盘下载文档在本地
3. 从云盘上删除文档
4. 后台系统实现对文档的日常基本操作(比如增删改文档)
5. 可以对文档的分门别类管理(重命名，移动文档)
6. 能排序搜索项目/文档，文档庞大的时候需要自动分页
7. 后续其他功能扩展
7.1 在线阅读：用户不需要下载到本地阅读(TBD)
7.2 安全管理，用户权限管理(TBD)
7.3 批量上传/下载文档(TBD)
7.4 导出文件列表到CSV表格(TBD)

4. 项目管理页面(增加/修改/删除)
5. 文档管理页面(增加/修改/删除/下载)
6. 上传文件页面(可以弹窗，表单提交等等tbd)
7. 用户登录/注册界面
8. ajax文档排序刷新页面
8. ...


二、后端（mysql数据库+php实现）
1. 本地环境搭建(AMP: apache+,mysql+php)
2. mysql数据库表格设计
3. thinkphp框架学习,七牛云存储研究
4. 创建/修改/删除项目条目(基本属性应包含名称, 上传时间，修改时间，简单描述等)
5. 添加/修改/删除文档条目(文档属性应包含名称，上传时间，修改时间，简单描述等)到项目
6. 在文档分类中移动文档
7. 上传一个本地文档到云盘
8. 从云盘上下载一个文档到本地
9. 实现用户登录功能
10. 用户注册功能

13. ...
